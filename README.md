# 📝 Todo List App

A simple Todo List web application built using **HTML, CSS, and JavaScript**.  
This project helps users add, manage, and delete daily tasks easily.

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed (if you added this feature)
- 💾 Data stored in browser (if you used localStorage)
- 📱 Responsive design (works on mobile & desktop)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
