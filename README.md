# 📝 Todo List App

A simple Todo List web application built using **HTML, CSS, and JavaScript**.  
This project helps users add, manage, and delete daily tasks easily.

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- 📱 Responsive design (works on mobile & desktop)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 📌 Project Purpose
